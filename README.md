## Week 1 – Static Mobile Store Website
Created a basic static website for an online mobile store. The main page links to separate pages for individual mobile products. Each product page includes:
* Product images
* Bulleted lists for mobile features and specs
* An embedded product demo video

## Week 2 – Layout with Nested Frames & CSS
Designed a homepage layout using nested frames. The page is split into three sections:
* **Title Frame:** Displays the name of the organization
* **Navigation Frame:** A sidebar containing links to other pages
* **Content Frame:** The main area where the linked pages open

Used **Inline**, **Internal**, and **External CSS** to style and improve the look of the webpages.

---

## Week 3 – Job Registration Form
Built a detailed Job Registration Form using HTML form components to collect user data. The form collects:
* Name and Gender
* Languages Known
* Contact Info (Email and Phone Number)
* Permanent Address and Preferred Job Location
* A short applicant description and a file upload field for CVs

Includes functioning **Submit** and **Reset** buttons.

---

## Week 4 – Bank Loan EMI Calculator
Developed a front-end loan application form that takes customer details like Name, Occupation, Loan Amount, Interest Rate, and Tenure (in years). 

Used **JavaScript** to calculate the Equated Monthly Installment (EMI) and display the result immediately after the form is submitted.

---

## Week 5 – Form Validation using JavaScript Regex
Added client-side validation to the Week 3 Job Registration Form using **JavaScript Regular Expressions (Regex)**. 

The script checks the following inputs:
* **Name** (Proper text format)
* **Email Address** (Valid email format)
* **Contact Number** (Correct digit length)

Shows clear error messages if the input is incorrect, and alerts **"Registered Successfully"** only when all fields are filled out properly.
